# Universal Screens — for Claude

## Suite docs — read before you start

This repo is one part of the **Universal Simulation suite**. Before starting work,
read the shared docs repo for cross-product context — how auth, the `@unisim/sdk`,
edge routing, and the suite changelog wire together:

- **Docs repo:** `universal-simulation-ltd/docs`, checked out at the umbrella root
  as `Docs_UNI_SIM/` (clone it there if it isn't present).
- Start with **`Docs_UNI_SIM/ARCHITECTURE.md`** (the cross-repo map) and
  **`Docs_UNI_SIM/claude-handover.md`** (latest session state + landmines).

When you finish user-visible work, add a suite **changelog** entry — see
["Pushing the changelog"](https://github.com/universal-simulation-ltd/docs/blob/main/README.md#pushing-the-changelog)
in `Docs_UNI_SIM/README.md`.
